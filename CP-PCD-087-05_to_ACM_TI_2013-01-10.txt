Add the following to the end of Section 3.Z.7.4 OBX-4 Observation Sub-ID
(ST) 00572, line 1515

The FACET identified items have correspondence values in their associated
OBX-3 Observation Identifier.  Refer to the OBX-3 Value column in the table
below for the value of OBX-3.  For identification of physiological
(numeric) alarms versus technical (non-numeric) alarms refer to the value
of OBX-8 Abnormal Flags for the Event Identification Facet.  If the message
does not contain an alarm type identification then it is assumed to be a
physiological alarm.


Add the following new OBX-3 Value column to Table 3.Z.7.4-1: Observation
Sub-ID Facets, line 1520

associated MDC value for the alarm.

EVENT_PHASE^EVENT_PHASE.
